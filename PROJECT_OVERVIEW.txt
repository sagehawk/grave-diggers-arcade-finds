Project Title: Grave Digger's Arcade Finds: A Modern Indie Game Discovery Platform

Project Summary:

Grave Digger's Arcade Finds is a feature-rich web application designed to be a one-stop destination for indie game enthusiasts to discover, explore, and engage with a vast library of indie games. The platform provides a modern, intuitive, and visually appealing user interface for browsing games, complete with detailed information, high-resolution screenshots, video trailers, and direct links to purchase the games from various online stores. The project is built with a cutting-edge frontend stack, leveraging the power of React, TypeScript, and the comprehensive RAWG Video Games Database API to deliver a dynamic, responsive, and engaging user experience.

Project Goals:

*   **Provide a centralized platform for indie game discovery:** To create a single place where users can find a wide variety of indie games, from popular new releases to hidden gems.
*   **Deliver a rich and engaging user experience:** To create a user interface that is not only functional but also visually appealing and enjoyable to use.
*   **Showcase modern frontend development practices:** To build a high-quality application that demonstrates a mastery of modern frontend technologies and best practices.
*   **Build a portfolio-worthy project:** To create a project that can be used to showcase my skills and experience to potential employers.

Target Audience:

The target audience for Grave Digger's Arcade Finds is indie game enthusiasts of all ages and backgrounds. This includes:

*   **Casual gamers:** Players who are looking for new and interesting games to play in their spare time.
*   **Hardcore gamers:** Players who are passionate about indie games and are always on the lookout for the next big thing.
*   **Game developers:** Developers who are looking for inspiration and want to see what other indie developers are creating.

Design and UX:

The design of Grave Digger's Arcade Finds is clean, modern, and visually appealing. The color scheme is dark and immersive, with a focus on highlighting the beautiful artwork of the games. The user experience is designed to be intuitive and easy to use, with a clear and consistent navigation structure. The application is also fully responsive, ensuring a seamless experience on all devices.

Challenges and Solutions:

*   **Challenge:** The RAWG API has a default limit of 20 results per request, which was not enough to provide a good infinite scrolling experience.
    *   **Solution:** I added a `page_size` parameter to the API requests to fetch up to 100 games at a time, providing a much larger pool of games for the user to discover.
*   **Challenge:** The RAWG API does not provide a way to exclude games with adult content.
    *   **Solution:** I implemented a client-side filtering solution that removes games with adult-themed keywords in their title, description, or genres. This ensures a safe and appropriate experience for all users.
*   **Challenge:** Hiding game cards for games without images was leaving gaps in the grid layout.
    *   **Solution:** I moved the filtering logic from the `GameCard` component to the `useGameFiltering` hook. This ensures that the games are filtered out before they are rendered, preventing any gaps in the grid.
*   **Challenge:** A subtle bug in the `useMemo` dependency array was preventing the filters from updating correctly.
    *   **Solution:** I identified and fixed the bug by correcting the dependency array, ensuring that the filters are always applied correctly.
